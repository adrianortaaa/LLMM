figure

Usa <figure> con <figcaption> Para imagenes que necesitan un título y una agrupacion semantica

<figure>
  <img src="Fotos/FigureImagen" alt="Imagen de Ejemplo">
  <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>

<output>

Contenedor para el resultado de un cálculo o la acción de un usuario en un formulario.

<form oninput="result.value = a.valueAsNumber * 2">
  <input type="number" id="a" value="10">
  X 2 =
  <output name="result" for="a">20</output>
</form>

<picture>

Selección de imagen adaptable (o responsive) usando <source>.

<picture>
  <source media="(min-width:800px)" srcset="image-large.jpg">
  <img src="image-small.jpg" alt="Responsive image">
</picture>
