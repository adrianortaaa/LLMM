figure

Use <figure> with <figcaption> for images that need a title and semantic grouping.

<figure>
  <img src="Fotos/FigureImagen" alt="Example Image">
  <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>

details / summary

Native disclosure widget: great for progressive enhancement and optional content.

<details>
  <summary>More info</summary>
  <p>Hidden content revealed on click.</p>
</details>

<output>

Container for the result of a calculation or user action in a form.

<form oninput="result.value = a.valueAsNumber * 2">
  <input type="number" id="a" value="10">
  X 2 =
  <output name="result" for="a">20</output>
</form>

<picture>

Adaptive (responsive) image selection using <source>.

<picture>
  <source media="(min-width:800px)" srcset="image-large.jpg">
  <img src="image-small.jpg" alt="Responsive image">
</picture>
