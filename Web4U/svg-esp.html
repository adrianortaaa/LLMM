<path>

El elemento más versátil, se utiliza para dibujar formas complejas y curvas mediante comandos (como mover, línea y curva).

<svg height="210" width="400" xmlns="">
       <path d="M150 0 L75 200 L225 200 Z" fill="orange" stroke="black" stroke-width="3" /> </svg>

<circle>

  <circle cx="60" cy="60" r="30" fill="#e44" />
</svg>

<image>

The element is used to insert an image in SVG.

<svg height="200" width="300" xmlns="http://www.w3.org/2000/svg"> 
  <image height="200" width="300" href="pulpitrock.jpg">
</svg>

<text>

Texto dentro de un documento SVG.

  <text x="5" y="15" fill="red">I love SVG!>Sorry, your browser does not support inline SVG.</text>

<rect>

El elemento <rect> se utiliza para dibujar rectángulos, definiendo su posición, ancho, alto y opcionalmente esquinas redondeadas.

<svg height="200" width="400" xmlns="http://www.w3.org/2000/svg">
    <rect x="50" y="40" width="300" height="120" fill="lightgreen" stroke="black" stroke-width="3" rx="15" ry="15"/>
</svg>
